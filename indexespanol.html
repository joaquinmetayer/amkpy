AMKpy es un script en el lenguaje de python donde se pueden automatizar tareas que se realizan con el mouse y el teclado como:

  • Click
  • Doble click
  • Seleccionar
  • Arrastrar
  • Scroll
  • Escribir texto
  • Comandos de teclado

Y todo lo que tu imaginación te permita… por ejemplo dibujar!

Agregando y modificando líneas de código dependiendo de la acción que quieras automatizar, ejecutando el código cada vez que necesites este proceso.

Guia

Requisitos:

  • Conocimientos básicos de programación / python.
  • Python instalado.

Pasos:

- Importar libreria pyAutogui, colocando dentro de consola:

MacOS pip3 install pyautogui
Windows pip install pyautogui

Encuentra mas sobre PyAutoGUI aquí.

- Descargar archivos aquí.

- Abrir main.py para calcular el tamaño de tu pantalla en pixeles para colocar nuestros click o acciones usando los ejes X e Y o alto y ancho. Este paso no es necesario pero es de ayuda para saber donde nos colocamos dentro de nuestra pantalla.

- Sabiendo lo anterior estamos listos para automatizar nuestra tarea, nuestro código sin habilitar ninguna función lo veremos así:

Dependiendo de lo que queramos hacer, añadiremos una de estas opciones debajo de la otra:

- pyautogui.click(70, 40) para el click.
- pyautogui.typewrite('writing hello') para escribir.
- pyautogui.hotkey('command', 'q') para atajos de teclado.
- pyautogui.scroll(-500) para scroll abajo.
- pyautogui.scroll(500) para scroll arriba.
- time.sleep(2) con el numero de segundos que quiero que se coloque en pausa.

Cuatro opciones, hacer clic, escribir, tecla de acceso directo y pausa... simple verdad?

Cuando agregue un comando y desee verificar o ejecutar el script, ¡ejecute el archivo! Aprende fallando.

No necesitas más para automatizar procesos, ten en cuenta que si necesitas automatizar algo más complejo, esta herramienta NO es para eso, ya que sabemos que existen otras formas más robustas de hacerlo, como el web scraping, entre otras.
